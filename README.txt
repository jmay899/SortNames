Note: This program needs python installed and present

Instructions on running:
1. Once files are on system from git, navigate to the directory they are stored in
2. Run "python sort.py" to sort normally, or "python sort.py -r" to sort in reverse order.

Instructions on testing:
1. Once files are on system from git, run appropriate test file
	a. If on Windows, run test.bat. Otherwise, run test.sh.
